## Notes

- Recommended image format: JPG at 90–95% quality
- Recommended image size: 1920×1080px (work/exploration), 1200×1200px for square
- Files are served directly from `/public` — they map to `/media/...` URLs in the browser
- If a file is missing the site shows a dark placeholder automatically
